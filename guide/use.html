# 如何使用该仓库

# 🎉 克隆项目

克隆项目后根据课程内容切换到相应的学习分支

git clone https://github.com/ziyi2/micro-framework.git

# ➕ 安装依赖

npm install

# 🚨 运行 ESLint

npm run lint

# 🚨 运行 Prettier

npm run prettier

# ✅ 执行单元测试

会在 coverage 目录下生成测试覆盖率报告

npm test

# 📝 执行 cz

使用 Angular 规范进行代码提交

npm run cz

# 🔊 生成日志

会在项目 docs 目录下生成 CHANGELOG.md 日志文件

npm run changelog

# 📦 构建库包

会在build目录下生成平级目录结构的API使用包

npm run build

# 📦 发布版本

npm run release

# 🚧 预览文档

启动静态网站

npm run docs:dev

# 🔖 生成文档静态资源

会在 .vuepress/dist 目录下生成静态资源包

npm run docs:build

# 🚀 部署

生成静态HTML文件并更新到远程的gh-pages分支上

npm run deploy
