# 介绍

Micro Framwork 是[深入浅出微前端课程](https://juejin.cn/book/7258893482318626868)衍生出来的一个示例框架，该框架主要基于 iframe 进行隔离设计，提供：

- 快照 & iframe 隔离能力
- 预加载、预渲染等性能优化手段
- Vue & React 微前端解决方案

Micro Framwork 仅仅是一个示例框架，切勿将其投入生产使用，该框架旨在帮助大家了前端设计的各个方面，包括微前端知识、工程化知识、浏览器底层原理知识等等。
